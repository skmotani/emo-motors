# EMO Motors Website

Industrial website for EMO Motors by Meera Industries - precision induction motors for textile machinery.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## Brand Colors

- Accent (CTA): #D11F2F
- Dark: #303B47
- Background: #F1F2F3
- Surface: #FFFFFF

## Pages

- `/` - Home
- `/products` - Product Hub
- `/products/ie3-ie4-motors` - IE3/IE4 Motors
- `/products/vfd-compatible` - VFD Motors
- `/products/brake-motors` - Brake Motors
- `/applications/ring-twisters` - Ring Twisters
- `/applications/winding-machines` - Winding Machines
- `/applications/spinning-tfo` - Spinning & TFO
- `/engineering` - Engineering Hub
- `/quality-lab` - Quality Lab
- `/contact` - Contact/RFQ
